# Windows-12-Setup-Prank

A harmless fake Windows 12 setup prank built with Python and Tkinter.

This project simulates a dramatic "Windows 12 Setup" screen for April Fools fun.  
It does not install, modify, encrypt, delete, or change anything on the user's PC.

## Features

- Fake Windows-style setup interface
- Slow progress animation for suspense
- Fake activity logs and warning messages
- Fake restart screen before the reveal
- Final April Fools message
- Custom app icon support
- Packaged as a standalone `.exe` with PyInstaller

## Safety

This project is only a visual prank.

It does **not**:
- install software
- edit system files
- change the registry
- restart the computer
- harm the device in any way

## Requirements

- Python 3.10+ recommended
- Windows
- Tkinter
- PyInstaller for building the executable
